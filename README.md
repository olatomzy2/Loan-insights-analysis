### Project Overview

This project analyzes loan application data to uncover insights related to loan approval trends, customer risk profiles, and financial performance metrics. The dashboard was built using Power BI (.pbix file) and focuses on data-driven decision-making.

### Objectives

Analyze loan approval and rejection patterns

Identify high-risk vs low-risk customer segments

Evaluate key financial indicators (loan amount, interest rates)

Create an interactive dashboard for business insights

### Tools & Technologies

1. Power BI – Data visualization & dashboard creation

2. Excel / CSV Dataset – Data source

3. Data Cleaning & Transformation – Power Query

4. DAX – KPI calculations and measures

### Key Insights

* Installment and Revolving Balance trends 

* Loan distribution by Loan Date

* Risk segmentation by credit history

* Total Deptors by Purpose of loan

* Default Rate indicators

### Dashboard Features

KPI Cards (Total Deptors, Default Rate%, Avg Interest Rate%, Avg FICO Score)

Loan Purpose Distribution Chart

Total Deptors vs Purpose of loan Analysis

Risk Category Segmentation

Interactive Filters (Year, Month)

### Business Impact

This dashboard helps:

Financial institutions assess risk faster

Improve loan approval decision processes

Identify profitable customer segments

Support strategic financial planning
